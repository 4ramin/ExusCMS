마야는 간단한 PHP 문자별 매치 프로그램입니다.
정규식을 사용하지 않아도 되는 간단한 문자열에 사용하면 됩니다.

 # 사용 : 

maya::execute(매치 법칙, 대상 문자열, 자료형, 디버그 여부=Boolean)

 # 파라미터 :

!문자열! : 오른쪽으로 해당 문자열만큼 완벽하게 매치되는 문자열을 찾습니다.
#문자열# : 오른쪽으로 문자열과 상관없이 비슷하게 매치되는 문자열을 찾습니다.
$문자열$ : 왼쪽으로 해당 문자열과 완벽하게 매치되는 문자열을 찾습니다.
+문자열+ : 매치 문자열의 모든 배열 앞에 추가할 문자열을 추가합니다.
@문자열@ : 문자열이 존재하는 마지막 위치를 찾습니다.

문자열||문자열 : 문자열을 배열로 지정합니다.
문자열^(int) : 문자열 뒤에 특정 길이의 문자열이 매치되는지 찾습니다.
^매치문자 : 매치문자를 일반 문자열로 지정합니다.

 # 추가 설명 : 

@\@+.+!jpg||png!

 1.\가 존재하는 맨 마지막 위치로 이동합니다.

\sample\text\my_folder\text\a.text
=>
\a.text (27)

 2.매치 문자열에 .을 붙입니다.

!jpg||png!
->
!.jpg||.png!

3.배열을 돌려서 해당 문자열이 오른쪽 맨 마지막에 존재하는지 확인합니다.
.jpg
text
FALSE

.png
text
FALSE

해당 함수에 FALSE를 반환합니다.
