Exuscms is a content management system (CMS)

Requirements
------------
- PHP 7.2 or higher.
- MariaDB/MySQL with PDO enabled
- Apache (recommended) or Nginx web server
- 80 MB of disk space (recommended), 50 MB (minimum)
