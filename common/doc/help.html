1.기본적인 요청

요청은 매우 간단합니다.


index.php는 /class/load.components.php를 인클루드시킵니다.

/class/load.components.php는 시스템 요구사항을 확인하고 클래스파일을 인클루드 시킨 후 세션을 시작하고 /class/base/base.php를 인클루드시킵니다.


base 클래스는 타임존을 설정하고 요청값을 초기화하고 /file/config/db.php파일이 존재하는지 확인후에 설치화면으로 넘기거나 데이터베이스, 스핑크스, 엘라스틱 서치를 초기화합니다.

초기화가 되었다면 {CMS 기본위치}/base/init/base.class.php를 인클루드시킵니다.


__construct는 model.class.php를 인클루드하고 base 클래스는 base/init/base.class.php는 init 함수를 요청합니다.

토큰값을 발급하고 GET 파라미터를 필터링합니다.

base 클래스에 저장되있는 정적 요청값을 받아온 후 GET/POST를 확인합니다.

GET/POST에 따라 클래스를 가져옵니다.


* GET은 base.class.php와 model.class.php를 인클루드합니다.

* POST는 controller.class.php와 model.class.php를 인클루드합니다.
